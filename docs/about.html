About Me

Hi, I’m Rutherford. Originally from the UK, currently based in The Hague, Netherlands. I’m a designer and developer primarily focused on type — in 2020, I founded Mass‑Driver, a studio for typeface design and font production.

About the Site

This site is a personal space for thoughts, side projects, and general reference. It’s part blog, part wiki, part digital garden. On a technical level, it’s designed to be minimal and accessible — it uses mostly semantic HTML with no JavaScript, and still works with CSS disabled.

The site is built from plaintext files by a custom tool, Keelback, and hosted as static HTML. Monospaced text is set in my own typeface, MD IO; the rest uses system defaults.
